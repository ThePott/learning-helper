## 파일 구조 방향 잡기

- src 안에는 pages, shared, features
- features에는 라우트별로 플랫하게 폴더 생성

```
book/           <- localhost:5173/book
book.create/    <- localhost:5173/book/create
book.delete/    <- localhost:5173/book/delete
```

- 각 features 폴더 안에선 기능별 폴더 분리

```
book/page/
book/type/
book/loader/
book/pending/
book/store/
(등등)
```
